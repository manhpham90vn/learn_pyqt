# PyQT

## Freeze

```shell
pip freeze > requirements.txt
```

## Install

```shell
pip install -r requirements.txt
```

## Start designer

```shell
pyqt6-tools designer
```

## Convert UI to Python

```shell
pyuic6 -o src/main.py -x ui/main.ui
```

## Start project

```shell
python src/main.py
```
